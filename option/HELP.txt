Option
--help	Display help guide.
--version	Displays current version.

Command
exit		Exit Software.

Qualifier
pi		Treat as pi.

HOW TO USE
[1] [operator] [2] [operator] ...

Available operators
"+" "-" "*" "/" "="

Caution
Please put a space between the number and the operator.
Supports integer and decimal point only calculations.
